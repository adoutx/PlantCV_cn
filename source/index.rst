 # PlantCV说明文档


## 介绍

[PlantCV](http://plantcv.danforthcenter.org/) 是应用于多种植物成像的功能模组。文档中将描述功能的使用方法以及如何在图像处理流程中应用。最初发布的PlantCV版本主要用来进行可见光（VIS）、近红外（NIR）以及激发荧光成像（PSII）的图像处理。PlantCV的开发还在不断迭代，我们希望得到更多植物表型组社区的助力。如有疑问或者建议，请在[GitHub issues page](https://github.com/danforthcenter/plantcv/issues)上提问。

注：在丹佛斯研究中心我们对内一般用FLU代指激发荧光成像(PSII)。但之前有已发表文章中，使用FLU代指固定态荧光成像系统（另一种荧光成像）。我们正在将FLU替换为PSII来避免可能导致的误解。
